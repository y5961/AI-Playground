📂 מבנה התיקייה
host.py / host_new.py: המנוע המרכזי (Orchestrator) שמנהל את השיחה, מחבר את הכלים ומתקשר עם Gemini.  

weather_Israel.py: שרת MCP המספק נתוני מזג אוויר עבור ערים בישראל.  

weather_USA.py: שרת MCP המספק נתוני מזג אוויר עבור ערים בארה"ב.  

client.py: מחלקת הבסיס לניהול התקשורת מול שרתי ה-MCP.  

.env: קובץ הגדרות המכיל את מפתח ה-API של Google.  

🛠️ טכנולוגיות
Python 3.12+

Google GenAI SDK: לניהול מודל Gemini 2.5 Flash.  

MCP Protocol: לאינטגרציה חלקה בין המודל לכלים חיצוניים.  

Asyncio: לניהול פעולות אסינכרוניות וריבוי משימות.

🤖 יכולות המערכת
זיהוי שפה אוטומטי: המערכת מזהה אם המשתמש פונה בעברית או באנגלית ומשיבה בהתאם (או לפי הגדרות המערכת).  

חיבור דינמי לכלים: המודל יודע לקרוא לפונקציות מתוך קבצי ה-Weather השונים בהתאם לצורך.  

חיפוש משלים: במידה והמידע לא קיים בשרתים המקומיים, המערכת יכולה להשתמש ב-Google Search כדי להשלים נתונים.
